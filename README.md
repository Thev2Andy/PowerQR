# PowerQR
PowerQR is an image data storage system, and can encode an unlimited amount of bytes in an image, *typically* compressing it in the process.

# Features
* Generate with one function.
```cs
QR.Generate(Bytes);
```

* Read with one function.
```cs
QR.Read(Image);
```
